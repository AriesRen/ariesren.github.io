## Markdown

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档，然后转换成丰富的HTML页面。

Markdown用一些简单的符号标识不同的标题，将某些文章标记为"**粗体**"或者*斜体*，下面就来一起学习一下。


<br>
## 语法

**1、标题**

不同的标题采用不等个数的`#`号来进行标记，如下所示：

```Markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
```

**2、代码块**

在需要高亮的代码块的前一行及后一行使用三个反引号“\`”，同时第一行反引号后面表面代码块所使用的语言, 如下：

\`\`\`pyhton
print ("Hello World!")
\`\`\`

**3、特殊字符**

```markdown
**粗体**
*斜体*
> 引用内容
```
